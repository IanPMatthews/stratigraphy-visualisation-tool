About this tool

This application generates stratigraphic column diagrams using Troels–Smith lithological symbols and Munsell colours. It is designed to display a column of soft sediment core data used in palaeoclimatic, palaeoenvironmental, and sedimentological, invesitigations. The figure may be downloaded in .svg format and edited using any suitable vector graphics tool.
