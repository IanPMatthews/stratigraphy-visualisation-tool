About this tool

This application generates stratigraphic column diagrams using Troels–Smith lithological symbols and Munsell colours. It is designed to display a column of soft sediment core data used in palaeoclimatic, palaeoenvironmental, and sedimentological investigations.

The figure may be downloaded in .svg format and edited using any suitable vector graphics tool.

To specify diffuse changes between units the upper and lower boundaries must be complementary i.e. a diffuse lower boundary in unit 1 must be matched by a diffuse upper boundary in unit 2.

If no single lithostratigraphy dominates the unit, symbols can be overlain. To ensure these do not obscure lower layers you must check the 'No background Fill' option.
